# 🎮 Stone Paper Scissors Game

A simple and interactive command-line Stone Paper Scissors game built using Python.
The game allows a user to play against the computer with multiple rounds, score tracking, and a smooth terminal experience.

---

## 🚀 Features

* 🎯 User vs Computer gameplay
* 🔄 Multiple rounds (best of 5)
* 📊 Live score tracking
* ⏳ Smooth processing animation using `time.sleep()`
* 🎨 Clean and readable CLI output
* 🔁 Play again option

---

## 🧠 Concepts Used

* Conditional statements (`if-else`)
* Loops (`while`,`for`)
* Lists
* Random module (`random.choice`)
* Time delay (`time.sleep`) for animations
* Basic user input handling

---

## ⚙️ How It Works

* The user enters a choice: `stone`, `paper`, or `scissors`
* The computer randomly selects a choice
* The result is calculated based on game rules
* Scores are updated after each round
* A short delay using `time.sleep()` is added to simulate processing and improve user experience

---

## ⏳ Animation Effect

The game uses:

```python
time.sleep()
```

to create:

* Processing delays (`⏳ Processing...`)
* Smooth output transitions
* A more engaging and realistic gameplay feel

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the file:

   ```
   game.py
   ```
